# مش بتتطبق تلقائيًا

الملفات هنا (`02-mssql-pv-pvc.yaml`, `03-mssql-deployment.yaml`) كانت بتشغّل MSSQL
جوه نفس الكلستر. اتشالت من `k8s/` الرئيسي لأن `kubectl apply -f k8s/` بيطبق ملفات
المجلد الرئيسي بس (مش بيدخل فولدرات فرعية)، فهي موجودة هنا للرجوع ليها بس.

دلوقتي الداتابيز بقت على **AWS RDS** (SQL Server Express - Free Tier) عشان توفر
RAM على الـ EC2 instance. لو حبيت ترجع للـ MSSQL المحلي تاني، انقل الملفين دول
لفولدر `k8s/` الرئيسي تاني.
